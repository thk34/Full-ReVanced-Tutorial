Full ReVanced Tutorial for Windows 10

In this tutorial you will learn how to compile revanced, install it and keep it up to date.

This tutorial is only for android phones, if you have an iPhone, just go buy a real phone.

Here are the prerequisites and their download links:

Azul Zulu OpenJDK 17: 

https://www.azul.com/downloads/?package=jdk#download-openjdk

Download the .msi version as it is easier to use

(Select java version 17, for the os select windows for the architecture select your cpu's architecture (usually 64-bit ) and for the package select JDK)

GitHub CLI:

https://cli.github.com/ (just click on download for Windows)

create a github account and in cmd run gh auth login  to login to your github account, it's obligatory if you want to stay up to date

The latest version of Android platform tools (only for if you plan to use the root version of revanced or to use the version that installs youtube revanced or to install YouTube revanced (or any other apk file) via adb):

https://developer.android.com/studio/releases/platform-tools (scroll to downloads and select the windows version)

the tutorial about adding it to path etc. is here.

Vanced MicroG installed on the phone (only for non root): it is in the files that is in the tutorial

To learn how to install an APK file, click here.

The tutorial:

1. Create a new folder, give it the name you want

2. Download this file

3. Extract the zip file's content to the folder

4. Run update.bat, it will download the latest version of all the files that are required to compile revanced.

5. Plug your phone to the pc if you plan to use a version that directly installs YouTube ReVanced on the phone.

6. Run the right compiling bat file:

  • If your phone is rooted and want to replace the original youtube app, then run compile-and-install-for-root.bat don't forget to right click, edit and and replace phone-id with the phone's id 
  • If your phone is not rooted or if you don't want to replace the original YouTube App and if you want the app to be installed right after the compilation, then run compile-and-install-for-non-root.bat
  • If your phone is not rooted or if you don't want to replace the original YouTube App and if you don't want the app to be installed right after the compilation and just want to have the final apk file to install it on your phone by sending it to your phone from your pc or just want to send the final apk to someone, then run compile-for-non-root.bat

To update:

Just run update.bat and compile it and install it with the right program using the criterias for choosing the right program that are above.
